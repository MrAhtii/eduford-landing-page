# EduFord University Website

A premium, fully responsive multi-page university website built using standard HTML5 and semantic CSS3. The project features a clean modern aesthetic, floating transparent navigation setups, parallax-style fixed backgrounds, and fluid layouts optimized across desktop and mobile viewing grids.

site link: []

## 🚀 Features

* **Multi-Page Architecture:** Includes fully designed structural pages: `index.html`, `courses.html`, `about.html`, `blog.html`, and `contact.html`.
* **Unified Stylesheet System:** Centralized `style.css` asset pipeline managing components globally for seamless loading speeds.
* **Dynamic Visual Layouts:** Full-screen hero headers using linear-gradient image overlays with fixed background attachments.
* **Intuitive Navigation Tracking:** Custom script-free visual indicators (`.active`) built right into the markup to highlight current directory positions.
* **Functional Contact Workflow:** Ready to deploy contact form structure integrated for standard post handler processing.
* **Fully Responsive Engine:** Custom CSS media queries ensuring optimized fluid break-points for mobile viewport resolutions.

⚙️ Setup and Installation
Run Locally:

Open index.html directly in any web browser.

Alternatively, run using the Live Server extension inside VS Code for real-time local hosting updates.

📝 Configuration Note (Contact Form)
The contact form embedded within contact.html is built as a front-end module. To receive user inputs directly into your email without setting up an external backend database infrastructure, update the action layer using Formspree:

HTML
<form action="(https://formspree.io/f/mgojvygq)" method="POST">
